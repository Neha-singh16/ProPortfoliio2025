# Achievement Images Guide

Add your achievement images to this directory with these specifications:

## Files to add:

1. **ideathon.jpg** - Delhi AI Grind Ideathon
   - Size: 1200px × 500px (16:7 aspect ratio recommended)
   - Format: JPG or PNG (will be optimized to WebP automatically)
   - Suggestion: Group team photo, award ceremony, or ideation session screenshot

2. **sports.jpg** - Three-Legged & Relay Race
   - Size: 1200px × 500px (16:7 aspect ratio recommended)
   - Format: JPG or PNG (will be optimized to WebP automatically)
   - Suggestion: Action shot during race, medal display, or team celebration photo

3. **travel.jpg** - Travel Platform Project
   - Size: 1200px × 500px (16:7 aspect ratio recommended)
   - Format: JPG or PNG (will be optimized to WebP automatically)
   - Suggestion: Dashboard screenshot, app mockup, or travel-related visual

## Image Optimization Tips:

- Use high-quality source images (1200x500px minimum)
- Compress images before adding (reduce file size)
- Use Vercel's Image Optimization (Next.js Image component handles this)
- Next.js will automatically serve WebP format to supported browsers

## Quick Image Sources:

- **Ideathon**: Use your actual team photo from the event
- **Sports**: Use an action/celebration shot from the races
- **Travel**: Create a mockup using Figma or take a screenshot of your app

All images will be automatically optimized and lazy-loaded by Next.js!
